# Collage Admission Management System

A PHP-based web application to manage student admissions for a college.  
This project aims to streamline the application process, manage courses & students, and simplify administrative workflows.

## Features

- Manage student registration and admission workflows  
- Admin panel to view, approve or reject applications  
- Course management (create, update, list courses)  
- Student list, details and status tracking  
- Authentication for admin and users  
- Responsive UI (if implemented)  
- Data storage using MySQL (or similar)  

---

## Technologies

- **Backend**: PHP  
- **Database**: MySQL (or MariaDB)  
- **Frontend**: HTML, CSS, JavaScript (basic)  
- **Version Control**: Git & GitHub  
- **Server Environment**: Local (e.g., XAMPP, WAMP, LAMP)
